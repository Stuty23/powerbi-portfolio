# Inventory Data Analysis

## Overview
This project focuses on analyzing inventory data by integrating multiple data sources and building interactive dashboards to track performance and identify inefficiencies.

## Tools & Technologies
- SQL Server
- MySQL
- Power BI
- DAX (Data Analysis Expressions)

## Project Workflow
- Imported inventory and product datasets into SQL Server and MySQL
- Performed data transformation and joined tables to create a unified dataset
- Loaded the processed data into Power BI for analysis
- Built interactive dashboards to monitor inventory performance
- Switched data sources between SQL Server and MySQL without affecting the report
- Maintained consistent table and column structure to ensure smooth transition between environments

## Key KPIs
- Average Demand per Day
- Average Availability per Day
- Total Supply Shortage
- Total Profit
- Total Loss
- Average Daily Loss
