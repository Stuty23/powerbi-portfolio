# Panic Attacks Data Analysis

## Overview
This project analyzes behavioral health data to identify patterns in panic attacks, symptoms, and lifestyle factors.

## Tools Used
- Snowflake
- Microsoft Power BI

## Key Features
- Integrated dataset from Snowflake into Power BI for analysis
- Performed data transformation and created calculated columns for categorization
- Built dashboards to analyze symptoms, triggers, and patient behavior
- Used DAX measures to generate percentage-based insights and comparisons
